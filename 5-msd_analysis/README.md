# Module 5: Biophysical Dynamics and MSD Analysis

This module provides the customized Python-based computational pipeline for extracting, quantifying, and fitting the Mean Squared Displacement (MSD) of subcellular organelles from D²-SIM super-resolution time-lapse sequences. 

Specifically, these scripts are configured to strictly reproduce the quantitative biophysical analyses and vector graphics presented in **Figure 4b** and **Figure 5e** of the main manuscript.

## 📂 Folder Contents

* `4b.py`: Python script designed to execute the dynamic tracking analysis and generate the corresponding plots for **Figure 4b**.
* `5e.py`: Python script configured to compute the anomalous diffusion regimes, perform non-linear curve fitting, and generate the MSD dynamics plot for **Figure 5e**.
* `D2-SIM.tif`: A representative D²-SIM reconstructed spatial-temporal image stack (Input data) utilized for the trajectory extraction.
* `Figure_5e_Mito_MSD_Dynamics.pdf`: The expected high-resolution vector graphic output generated by `5e.py`.

## ⚙️ Key Biophysical Constraints

To ensure absolute quantitative fidelity and eliminate statistical or optical artifacts, the analysis scripts rigorously implement the physical models detailed in **Supplementary Note 4: : Derivation and fitting configuration of the Mean Squared Displacement**:
1. **25% Truncation Rule**: The MSD curve fitting is strictly constrained to the first 25% of the total trajectory time lags ($\Delta t$) to mathematically prevent statistical bias and divergence arising from finite biological trajectory lengths.
2. **Optical Localization Error Compensation**: The analytical anomalous diffusion model explicitly incorporates a $4\sigma_{loc}^2$ compensation term. This strictly decouples the authentic biophysical diffusion dynamics from the intrinsic optoelectronic localization uncertainty of the D²-SIM architecture.
